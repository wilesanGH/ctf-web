    Vulhub 靶场搭建指南¶

    Vulhub 是一个基于 Docker 和 docker-compose 的漏洞环境集合，包含了大量真实漏洞环境，可用于漏洞复现和学习。

    1. 环境准备¶

    1.1 安装 Docker¶

    访问 Docker 官网 下载 Docker Desktop

    根据系统提示完成安装

    启动 Docker Desktop

    打开终端，验证安装：

    docker --version

    1.2 安装 Docker Compose¶

    Docker Desktop for Windows/Mac 已经包含了 docker-compose

    Linux 用户需要额外安装：

    sudo apt install docker-compose  # Ubuntu/Debian
sudo yum install docker-compose  # CentOS

    1.3 获取 Vulhub¶

    克隆官方仓库：

    git clone https://github.com/vulhub/vulhub.git

    2. 使用方法¶

    2.1 基本操作¶

    进入漏洞目录： 

    cd vulhub/flask/ssti

    编译并启动环境： 

    docker-compose up -d

    停止并删除环境： 

    docker-compose down

    2.2 常用命令¶

    查看运行状态： 

    docker-compose ps

    查看容器日志： 

    docker-compose logs

    重新构建环境： 

    docker-compose build

    4. 常见问题¶

    4.1 环境问题¶

    构建失败：

    检查网络连接

    更新 Docker 镜像

    清理 Docker 缓存

    启动错误：

    检查端口占用

    验证配置文件

    查看错误日志

    4.2 漏洞复现问题¶

    环境无法访问：

    确认容器状态

    检查端口映射

    验证网络配置

    漏洞利用失败：

    确认环境版本

    检查利用条件

    调整攻击参数

    5. 安全提示¶

    5.1 环境安全¶

    隔离措施：

    使用独立网络

    限制端口访问

    及时关闭环境

    5.2 使用建议¶

    合规使用：

    仅用于学习研究

    不要对外网开放

    遵守相关法律

    数据保护：

    不使用敏感数据

    定期清理环境
